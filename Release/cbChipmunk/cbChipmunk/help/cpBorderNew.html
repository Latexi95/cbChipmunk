cpBorderNew


   KUVAUS

Luo uudet reunukset laatikon muotoisen alueen ympärille. Tarkoitettu lähinnä esimerkkejä varten, sillä reunuksia ei (ainakaan tällä hetkellä) voi poistaa taikka muutenkaan muokata. Sopii parhaiten estämään törmäysmuotojen pääsyn pelialueen/ruudun ulkopuolelle.
   KÄYTTÖ
cpBorderNew(geometry,r# = 1,layer = CP_GROUP_ALL,group = CP_NO_GROUP,offsetVecX# = 0,offsetVecY# = 0,friction# = 0.7,restitution# = 0.3)

  • geometry = cpBoxGeometryNew:llä luotu laatikko, jolle reunukset tehdään.

  • r# = Reunuksien teossa käytettävien viivojen säde. Reunat ulottuvat viivan säteen verran ilmoitetun koon sisäpuolelle.

  • layers = Törmäyskerrokset, joihin reunat vaikuttavat. Oletuksena kaikkiin.

  • group = Törmäysryhmä johon reunukset kuuluvat.

  • offsetVecX# = Laatikon keskipisteen sijainti x-akselilla.

  • offsetVecY# = Laatikon keskipisteen sijainti y-akselilla.

  • friction# = Reunojen kitka.

  • restitution# = Reunojen kimmoisuus.
